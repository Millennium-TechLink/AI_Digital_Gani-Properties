- **Cost**: $0/month
- **Bandwidth**: 100GB/month
- **Builds**: Unlimited
- **Serverless Functions**: 100GB-hours/month
- **Edge Functions**: 100GB-hours/month

**What's Included:**
- ✅ Website hosting

- ✅ CDN
- ✅ Preview deployments

**When to Upgrade:**
- If you exceed 100GB bandwidth/month

- If you need advanced analytics

#### Pro Plan (If Needed)

**Upgrade When:**
- High traffic (>100GB/month)
- Need team collaboration
- Need advanced analytics

### 2. Database: Supabase

#### Free Tier
- **Cost**: $0/month
- **Database Size**: 500MB
- **Bandwidth**: 5GB/month
- **API Requests**: Unlimited
- **Realtime**: 200 concurrent connections
- **Storage**: 1GB

**What's Included:**
- ✅ PostgreSQL database
- ✅ Row Level Security (RLS)
- ✅ Auto backups
- ✅ API access
- ✅ Dashboard access

**Storage Estimate:**
- Properties table: ~50KB per property (100 properties = 5MB)
- Form submissions: ~1KB per submission (3,000 submissions = 3MB)
- **Total**: ~10-20MB (well within 500MB limit)

**When to Upgrade:**
- Database exceeds 500MB
- Need more storage (images)
- Need more bandwidth

#### Pro Plan (If Needed)
- **Cost**: $25/month
- **Database Size**: 8GB
- **Bandwidth**: 50GB/month
- **Storage**: 100GB
- **Backups**: 7-day retention

**Upgrade When:**
- Database > 500MB
- Storage > 1GB (many property images)
- Need longer backup retention

## 💡 Cost Optimization Tips

### 1. Stay on Free Tier Longer
- Optimize images (compress before upload)
- Use CDN for static assets
- Cache API responses
- Monitor usage regularly

### 2. Upgrade Strategically
- Only upgrade when you actually need it
- Monitor usage before upgrading
- Start with one service upgrade at a time

### 3. Alternative Free Options
- **Email**: Can use Web3Forms (250/month free) + Supabase
- **Storage**: Can use Cloudinary (25GB free) instead of Supabase Storage
- **Hosting**: Can use Netlify (100GB free) instead of Vercel
